# Ethereum Contract Monitor 

## What's this?
This is a simple monitor that watches for new contract creations on Ethereum Mainnet.


## How to use
1. Clone this repo 
2. `npm install` 
3. edit .env file with your own Alchemy HTTPS and WEBSOCKETS URL and Discord Webhook URL
4. `npm start`
